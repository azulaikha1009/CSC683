# Car Game

## Overview

Car Game is a beginner-friendly Construct 3 tutorial project that introduces the fundamentals of 2D game development. The project focuses on implementing player controls, collision detection, and simple gameplay mechanics.

## Technologies Used

- Construct 3

## Learning Objectives

- Understand Construct 3 interface
- Implement player movement
- Detect collisions between objects
- Create simple game mechanics
- Organize game layouts and event sheets

## Project File

- `CarGame.c3p`

## Course

CSC683 – Game Development
