# Roti Man

## Overview

Roti Man is a Construct 3 tutorial project designed to introduce character movement, object interaction, and basic gameplay mechanics in a 2D environment.

## Technologies Used

- Construct 3

## Learning Objectives

- Create player movement
- Implement object interactions
- Design simple game levels
- Apply collision detection
- Develop gameplay logic using event sheets

## Project File

- `RotiMan.c3p`

## Course

CSC683 – Game Development
