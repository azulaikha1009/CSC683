# Flappy Bird

## Overview

Flappy Bird is a Construct 3 tutorial project inspired by the classic side-scrolling game. The project demonstrates gravity-based movement, obstacle navigation, and collision handling.

## Technologies Used

- Construct 3

## Learning Objectives

- Implement gravity and jumping mechanics
- Create scrolling gameplay
- Detect collisions with obstacles
- Develop score-based gameplay
- Understand event-driven programming

## Project File

- `FlappyBird.c3p`

## Course

CSC683 – Game Development
