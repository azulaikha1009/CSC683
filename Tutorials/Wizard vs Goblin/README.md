# Wizard vs Goblin

## Overview

Wizard vs Goblin is a Construct 3 tutorial project that introduces action game mechanics, including player interactions, enemy behavior, and combat elements.

## Technologies Used

- Construct 3

## Learning Objectives

- Develop player movement
- Create enemy interactions
- Implement combat mechanics
- Manage game events
- Build a simple action game

## Project File

- `WizardVsGoblin.c3p`

## Course

CSC683 – Game Development
